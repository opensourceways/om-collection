# om-collections

Operation system for opensource community. such as: [openEuler](https://openeuler.org/zh/), [mindSpore](https://www.mindspore.cn/)

## Download
```
$ git clone https://gitee.com/opensourceway/om-collections
```

## Usage
```
 cd om-collections
 python  george.py
```
